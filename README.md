# ESP32-ADC-PWM-Project

# Smart Temperature Monitoring and Automatic LED controlling 

## Project Description
This project is a Automatic LED controlling with a combination of LDR and Potentiometer, also a real-time temperature monitoring.

The system displays:
1. Potentiometer value
2. LDR Real time value
3. LDR Lux
4. Real Time Temperature

---

# Components Used

- ESP32
- DHT11 Sensor
- Potentiometer
- LDR Module
- 220 ohms resistor
- Push Button
- Jumper Wires

---

# Pin Connections

| Component | ESP32 Pin |
|----------|----------|
| POTENTIOMETER PIN | GPIO 34 |
| LDR PIN | GPIO 34 |
| DHT PIN | GPIO 32 |
| LED PIN | GPIO 25 |

---

# Required Libraries

Install the following libraries in Arduino IDE:

1. DHT sensor library

---

# Installation Steps

1. Open Arduino IDE
2. Install required libraries
3. Connect ESP32
4. Upload the source code
5. Open Serial Monitor
